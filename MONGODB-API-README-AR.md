# تشغيل CASH TOP على أي استضافة بدون ملف rtdb.js

هذه النسخة لا تحتوي على `api/rtdb.js` ولا تعتمد على API داخل نفس استضافة الواجهة.

## طريقة العمل

- ملفات HTML/CSS/JS يمكن رفعها على أي استضافة ثابتة مثل GitHub Pages أو Netlify أو Vercel أو Firebase Hosting أو استضافة cPanel عادية.
- رابط خدمة MongoDB الخارجية موجود في `firebase-config.js` داخل `config.databaseURL`.
- الواجهة ترسل عمليات المزامنة إلى خدمة HTTPS الخارجية بغض النظر عن نطاق الاستضافة الحالية.
- بيانات اتصال MongoDB السرية وكلمة المرور يجب أن تبقى في الخادم الخارجي فقط، ولا توضع داخل ملفات المتصفح.

## الرابط الحالي

`https://cash-top-33.vercel.app/api/rtdb`

يمكن تغييره لاحقاً من `firebase-config.js` بدون تعديل بقية صفحات المشروع، بشرط أن تحافظ الخدمة الجديدة على نفس صيغة GET/PUT/PATCH/DELETE المستخدمة حالياً.

## مهم

هذه الحزمة Frontend محمولة، لكنها تحتاج أن تكون خدمة MongoDB الخارجية شغالة وتسمح بطلبات CORS من نطاق الاستضافة التي تستخدمها.
