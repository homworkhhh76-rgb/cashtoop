# Revision 42

- إضافة ترس إعدادات في صفحة الإدارة العامة لتغيير كلمة مرور المشرف العام مع مزامنتها.
- إضافة نسخ احتياطي شامل يجمع بيانات الإدارة وجميع بيانات الشركات، يحسب الحجم قبل التحميل، ويصدر ملف JSON مستقل.
- إضافة استيراد شامل مع شريط تقدم من 0% إلى 100% وإعادة رفع كل شركة إلى قاعدة البيانات الحالية.
- إضافة وضع صيانة عام من لوحة الإدارة، وتحديث وصول كل الشركات فوراً، وتحويل الجلسات إلى صفحة صيانة مستقلة ومتجاوبة.
- صفحة الصيانة تتحقق تلقائياً من حالة الشركة وتعيد المستخدم إلى تسجيل الدخول عند انتهاء الصيانة.
- تعزيز الخروج التلقائي عند إيقاف/حذف مفتاح الشركة أو تعطيل/حذف الموظف عبر بيانات الوصول والمزامنة السريعة الموجودة في النظام.
